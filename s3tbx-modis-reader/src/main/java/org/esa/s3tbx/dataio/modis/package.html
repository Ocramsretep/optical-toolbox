This module provides a product reader for the MODIS product formats. The current implementation allows for importing of all MODIS L1b product types of bot the AQUA and TERRA sensor. The supported types in detail are:
- MOD021KM
- MYD021KM
- MOD02HKM
- MYD02HKM
- MOD021KM_IMAPP
- MOD02QKM
- MYD02QKM
- MODOCL2
- MYDOCL2
- MODOCL2A
- MYDOCL2A
- MODOCL2B
- MYDOCL2B
- MOD28L2
- MYD28L2
- MODOCQC
- MYDOCQC
- MOD13A2
- MYD13A2
- MOD15A2
- MYD15A2
- MOD09GA
- MYD09GA
